# Customer Churn Analysis

## Overview
This project analyzes customer churn data to identify the key factors influencing customer retention and support data-driven business decisions.

## Tools & Technologies
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

## Project Workflow
- Data Cleaning
- Data Preprocessing
- Exploratory Data Analysis (EDA)
- Data Visualization

## Key Insights
- Analyzed customer behavior and churn patterns.
- Identified factors associated with customer churn.
- Created visualizations to support business insights.
